const char * OmErrorString (int status)

Detailed Description

API return and error codes.

The return values to most API functions are plain int types.

When successful, the functions return zero or a positive integer result. When unsuccessful, the functions return a negative integer result.

OmErrorString() returns a string representation of an error code.

Remarks:
An enumerated type is not used as the API return values because some functions return successfully with (non-negative) numeric results.
